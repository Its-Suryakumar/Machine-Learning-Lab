Model Intepretation

Why Naive Bayes for this dataset?

  • We are using Navie Bayes for this dataset as the datset is categorical and the independent variables (conditionally idependent) are unrelated with each other.

Classification report interpretation:

  • From the classification report we can see that the precision and recall for class "Not placed" is 0.75 and 0.90 respectively meaning that the 75% of the predictions made by the model is correct and 90% of the relevant data points were correctly identified. Similary for class "Placed" precision is 95% and recall is 87%. Means the preprocessed dataset is well trained and processed such that it yeilds about 95% precise outcome for classifying "Placed" and 75% average outcome for "Not Placed". The model is found to be highly accurate.
  • And as for the accuracy of the model it is 0.88 i.e., 88% of outcomes were predicted correct.
